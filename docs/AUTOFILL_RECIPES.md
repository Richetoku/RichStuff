# Autofill Recipes

This file documents the generated catch-up pass for the request that every RichStuff item have a recipe.

The pass added recipes under `data/richstuff/recipe/autofill/` only when an item had no existing recipe output. It prefers material-specific recipes for nuggets, tiny dusts, dust blocks, storage blocks, budding crystals, gem processing steps, molds, and machine frames. Generic fallback recipes are only used for leftover generated helper/icon items.
